# E-commerce Car Website

This is a simple e-commerce website for a car showroom, created using HTML and CSS.

## Pages:
- Home (`index.html`)
- Products (`products.html`)
- About Us (`about.html`)
- Contact Us (`contact.html`)

## Features:
- Stylish design with good UI/UX principles.
- Responsive layout (basic).
- Navigation bar.
- Hero section.
- Featured cars display.
- Contact form.

## Setup:
To view the website, simply open `index.html` in your web browser.
